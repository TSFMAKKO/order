# 圖片目錄說明

這個目錄包含網站所需的圖片文件。

## 產品圖片

請在 `products` 子目錄中放置以下產品圖片文件：

- `phone.jpg` - 智能手機圖片
- `laptop.jpg` - 筆記本電腦圖片
- `headphones.jpg` - 無線耳機圖片
- `shoes.jpg` - 運動鞋圖片
- `monitor.jpg` - 桌面顯示器圖片
- `smartwatch.jpg` - 智能手錶圖片

## 默認圖片

- `no-image.jpg` - 當產品沒有圖片時顯示的默認圖片

## 支付圖片

- `ecpay-logo.png` - 綠界支付標誌

## 注意事項

請確保所有圖片尺寸適中，建議產品圖片寬度統一為600-800像素，以獲得最佳顯示效果。
